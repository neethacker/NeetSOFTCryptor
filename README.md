# 🔒 Neet Cryptor

**Neet Cryptor** – это простой инструмент с графическим интерфейсом для шифрования и упаковки исполняемых файлов Windows (.exe).  
Он шифрует ваш файл с помощью XOR и создаёт загрузчик, который расшифровывает и запускает оригинальный EXE «на лету» без записи на диск (кроме временной папки).

> ⚠️ **Только для образовательных целей и тестирования в авторизованных средах.**  
> Не используйте для обхода антивирусных решений без разрешения владельца системы.

---

## ✨ Возможности

- 🖥️ Простой GUI на `customtkinter`
- 🔐 Шифрование исходного EXE (XOR с случайным ключом)
- 📦 Упаковка в один загрузчик (через PyInstaller)
- 🪟 Запуск расшифрованного файла без консольного окна
- 🎨 Возможность установить собственную иконку `.ico`
- 🧹 Никаких бэкдоров, DLL или скрытых нагрузок – только защита одного файла

---

## 📦 Требования

- Windows (7/10/11)
- Установленный **Python 3.10+** (добавлен в PATH)
- Установленный **PyInstaller**

pip install customtkinter psutil pyinstaller
